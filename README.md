# Caesar Cipher Encryption Tool – Project Description

The **Caesar Cipher Encryption Tool** is a Python-based application developed to demonstrate the fundamentals of classical cryptography. The project implements the Caesar Cipher algorithm, a simple substitution cipher in which each alphabetic character in a message is shifted by a user-defined number of positions (called the **shift value** or **key**) within the alphabet.

The application allows users to enter a message and choose a custom shift value to encrypt the text. It also provides a decryption feature that reverses the encryption process using the same key, allowing the original message to be recovered accurately. The program preserves uppercase letters, lowercase letters, spaces, numbers, and special characters, ensuring that only alphabetic characters are encrypted.

This project is designed using core Python programming concepts without relying on external libraries. It demonstrates the practical use of functions, loops, conditional statements, string manipulation, and ASCII character conversion using the built-in `ord()` and `chr()` functions. Through this project, users gain a basic understanding of how simple encryption algorithms work and how plaintext can be transformed into ciphertext for secure communication.

### Key Features

* Encrypts messages using the Caesar Cipher algorithm.
* Decrypts encrypted messages using the same shift key.
* Allows users to choose a custom shift value (key).
* Supports both uppercase and lowercase letters.
* Preserves spaces, numbers, and special characters.
* Displays the original, encrypted, and decrypted messages clearly.
* Developed using only built-in Python features without external libraries.

### Technologies Used

* Python 3
* Functions
* Loops (`for`)
* Conditional Statements (`if`, `else`)
* String Manipulation
* ASCII Character Functions (`ord()` and `chr()`)

### Learning Outcomes

This project enhanced my understanding of basic cryptography concepts, encryption and decryption techniques, ASCII character encoding, string manipulation, function implementation, and logical problem-solving using Python. It also provided practical experience in developing a simple command-line application for secure message transformation.
